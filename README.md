Auto-Checkout Program
===
This is a program contains various helper functions for Momo.
---

### Features ###
<ul>
    <li>Automatically fill out the checkout form.
    <li>Automatically put a certain item into cart and perform checkout at a specified time assgined by user.
    <li> Automatically login.
</ul>

### Prerequisites ###
<ul>
  <li> This app might only be compatible with MacOS.
  <li> Newest version of chrome driver: http://chromedriver.chromium.org
  <li> Guide for accessing "User-Agent" of your browser: https://www.searchenginejournal.com/change-user-agent/368448/
</ul>

### Quick start instructions
<ol type="1">
  <li> Clone the whole directory to your Mac.
  <li> Download chrome driver from the link provided in the Prerequisites section.
  <li> Access your User-Agent.
  <li> Use "python main.py" to run this program and follow the instructions.
</ol>

### Environments
OS: *MacOS, Catalina 10.15.7*   
chrome driver: *ChromeDriver 87.0.4280.88*  
python: *Python 3.7.3*
